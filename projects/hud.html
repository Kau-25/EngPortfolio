Firefighter Mask HUD

Embedded firmware · electronics design · web backend

Overview

A wearable heads-up display for a firefighter's mask — an OLED visor display showing GPS position and heading alongside a helmet-mounted camera, all tied into a command dashboard that lets incident commanders track every firefighter in real time.

What I built

  • Built a wearable heads-up display for a firefighter's mask: an ESP32 drives an OLED in the visor, paired with a GPS module and magnetometer for position and heading, plus a helmet-mounted ESP32-CAM streaming live video.
  • Designed and wired all of the electronics across both ESP32 nodes — my primary role on the team.
  • Wrote the Python web-server backend that ingests each firefighter's data and powers a command dashboard: pushing text alerts to the OLED HUD, plotting live location and heading on a mini-map, and showing every firefighter's camera feed.
