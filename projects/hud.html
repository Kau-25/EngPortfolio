Firefighter Mask HUD

Embedded firmware · electronics design · web backend

A wearable HUD for a firefighter's mask, with a command dashboard for incident commanders to track every firefighter in real time.

  • An ESP32 drives an OLED in the visor showing GPS position and heading; a helmet-mounted ESP32-CAM streams live video — I designed and wired all electronics across both nodes.
  • Wrote the Python web-server backend powering the command dashboard: pushing text alerts to the OLED, plotting live location on a mini-map, and displaying each firefighter's camera feed.
