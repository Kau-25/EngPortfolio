LLM-Controlled Autonomous Drone

Flight controls · onboard perception · natural-language command

Overview

Built and configured a quadcopter on a Matek F405-WMN flight controller running ArduPilot; calibrated sensors and flew an autonomous 3-waypoint mission inside a geofence.

What I built

  • Built and configured a quadcopter on a Matek F405-WMN flight controller running ArduPilot; calibrated sensors and flew an autonomous 3-waypoint mission inside a geofence.
  • Streamed live video from a Raspberry Pi Zero W to an AWS server running YOLO object detection over a Tailscale VPN for real-time onboard perception.
  • Wrote an MCP server letting an LLM interface with the flight controller over MAVLink — setting flight parameters and modes, and commanding arm, takeoff, and disarm.
