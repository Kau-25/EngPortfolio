LLM-Controlled Autonomous Drone

Flight controls · onboard perception · natural-language command

A 3-person build covering drone assembly, onboard vision, and natural-language flight control over MAVLink.

  • Assembled and configured the quadcopter on a Matek F405-WMN flight controller running ArduPilot; calibrated sensors and flew an autonomous 3-waypoint mission inside a geofence — my primary hardware role.
  • Wrote an MCP server letting an LLM interface with the flight controller over MAVLink — setting flight parameters and modes, and commanding arm, takeoff, and disarm — my primary software role.
  • Streamed live video from a Raspberry Pi Zero W to an AWS server running YOLO object detection over a Tailscale VPN for real-time onboard perception.
